# Rain & Wind Synthesizer

Original patch by **Simple Trax** & **Taavi Viikman**  
[Video Demo](https://youtu.be/ZXwk8sDts0Y)  

Hvcc adaptation by **Ledlaux**. 

This code creates USB MIDI device. Send MIDI CC messages to control generator parameters.

## MIDI Control (Channel 1)

| CC | Parameter    |
|----|-------------|
| 7  | Volume      |
| 10 | Crackles    |
| 11 | Drop Amount |
| 12 | Hi-Pass     |
| 13 | Intensity   |
| 14 | Wind        |
| 15 | Drop Gain   |
| 16 | Drop Size   |
| 17 | Size        |
